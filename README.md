# Student Grade Calculator

A simple and user-friendly program to calculate student grades based on their subject marks.

## Features
- **Input:** Accepts marks for multiple subjects.
- **Calculation:** Automatically calculates total score and percentage.
- **Grading:** Displays the final grade based on the percentage (e.g., A, B, C, F).
- **Simple:** Clean and easy-to-understand code.
